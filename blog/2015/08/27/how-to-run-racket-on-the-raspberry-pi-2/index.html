How to run Racket on the Raspberry Pi 2

Aug 27th, 2015 1:25 pm

I got a Raspberry Pi 2 Model B to play with. I was wondering how difficult it is to get Racket running on the Raspberry Pi. I downloaded the Unix source + built packages tarball from Racket’s homepage because I only wanted to compile the core of Racket. After unpacking the tarball I was suprised that the instructions where quite short:
